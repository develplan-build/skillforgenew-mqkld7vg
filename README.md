# SkillForgenew - B2B Course Management Platform

## What you received
The online site you see is a **demo** running entirely in the browser with local state. This repository contains the **complete, production-ready source code** for the application. To make it real, you need to connect your own services (Supabase, Stripe, SendGrid) by following this guide.

## Requirements
- Node.js 18+
- Accounts for: Supabase, Stripe, SendGrid, Google Cloud Console (for Google Login), Apple Developer (for Apple Login), Microsoft Azure (for Microsoft Login).
